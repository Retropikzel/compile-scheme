# Install

## Linux

First install Gauche Scheme, on Debian/Ubuntu

    apt-get install gauche

Then build and install compile-scheme

    make
    make install

## Mac OS

First install Gauche Scheme, on Debian/Ubuntu

    brew install gauche

Then build and install compile-scheme

    make
    make install

## Windows
To install on windows first install Gauche Scheme. Then run install.bat as
administrator.
